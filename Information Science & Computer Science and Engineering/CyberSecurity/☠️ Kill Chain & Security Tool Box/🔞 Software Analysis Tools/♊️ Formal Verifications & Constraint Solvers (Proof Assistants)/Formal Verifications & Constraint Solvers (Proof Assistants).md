## Intro
### Proof Assistants
> 🔗 https://en.wikipedia.org/wiki/Proof_assistant#System_comparison

In [computer science](https://en.wikipedia.org/wiki/Computer_science "Computer science") and [mathematical logic](https://en.wikipedia.org/wiki/Mathematical_logic "Mathematical logic"), a **proof assistant** or **interactive theorem prover** is a software tool to assist with the development of [formal proofs](https://en.wikipedia.org/wiki/Formal_proof "Formal proof") by human–machine collaboration. This involves some sort of interactive proof editor, or other [interface](https://en.wikipedia.org/wiki/User_interface "User interface"), with which a human can guide the search for proofs, the details of which are stored in, and some steps provided by, a [computer](https://en.wikipedia.org/wiki/Computer "Computer").

A recent effort within this field is making these tools use [artificial intelligence](https://en.wikipedia.org/wiki/Artificial_intelligence "Artificial intelligence") to automate the formalization of ordinary mathematics.
