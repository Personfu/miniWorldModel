## Intro
> 🔗 https://en.wikipedia.org/wiki/Term_algebra

In [universal algebra](https://en.wikipedia.org/wiki/Universal_algebra "Universal algebra") and [mathematical logic](https://en.wikipedia.org/wiki/Mathematical_logic "Mathematical logic"), a **term algebra** is a freely generated [algebraic structure](https://en.wikipedia.org/wiki/Algebraic_structure "Algebraic structure") over a given [signature](https://en.wikipedia.org/wiki/Signature_\(logic\) "Signature (logic)"). For example, in a [signature](https://en.wikipedia.org/wiki/Signature_\(mathematical_logic\) "Signature (mathematical logic)") consisting of a single [binary operation](https://en.wikipedia.org/wiki/Binary_operation "Binary operation"), the term algebra over a set _X_ of variables is exactly the [free magma](https://en.wikipedia.org/wiki/Free_magma "Free magma") generated by _X_. Other synonyms for the notion include **absolutely free algebra** and **anarchic algebra**.

From a [category theory](https://en.wikipedia.org/wiki/Category_theory "Category theory") perspective, a term algebra is the [initial object](https://en.wikipedia.org/wiki/Initial_object "Initial object") for the [category of all _X_-generated algebras of the same signature](https://en.wikipedia.org/wiki/F-algebra "F-algebra"), and this object, unique up to [isomorphism](https://en.wikipedia.org/wiki/Isomorphism "Isomorphism"), is called an [initial algebra](https://en.wikipedia.org/wiki/Initial_algebra "Initial algebra"); it generates by [homomorphic](https://en.wikipedia.org/wiki/Homomorphic "Homomorphic") projection all algebras in the category.

A similar notion is that of a _[Herbrand universe](https://en.wikipedia.org/wiki/Herbrand_universe "Herbrand universe")_ in [logic](https://en.wikipedia.org/wiki/Logic "Logic"), usually used under this name in [logic programming](https://en.wikipedia.org/wiki/Logic_programming "Logic programming"), which is (absolutely freely) defined starting from the set of constants and function symbols in a set of [clauses](https://en.wikipedia.org/wiki/Clause_\(logic\) "Clause (logic)"). That is, the Herbrand universe consists of all [ground terms](https://en.wikipedia.org/wiki/Ground_term "Ground term"): terms that have no variables in them.

An [atomic formula](https://en.wikipedia.org/wiki/Atomic_formula "Atomic formula") or _atom_ is commonly defined as a [predicate](https://en.wikipedia.org/wiki/Predicate_\(mathematical_logic\) "Predicate (mathematical logic)") applied to a tuple of terms; a [ground atom](https://en.wikipedia.org/wiki/Ground_atom "Ground atom") is then a predicate in which only ground terms appear. The _Herbrand base_ is the set of all ground atoms that can be formed from predicate symbols in the original set of clauses and terms in its Herbrand universe. These two concepts are named after [Jacques Herbrand](https://en.wikipedia.org/wiki/Jacques_Herbrand "Jacques Herbrand").

Term algebras also play a role in the [semantics](https://en.wikipedia.org/wiki/Semantics "Semantics") of [abstract data types](https://en.wikipedia.org/wiki/Abstract_data_type "Abstract data type"), where an abstract data type declaration provides the signature of a multi-sorted algebraic structure and the term algebra is a concrete model of the abstract declaration.


### Definition
> 🔗 https://en.wikipedia.org/wiki/Term_algebra#Universal_algebra

A **type** $\tau$ is a set of function symbols, with each having an associated arity (i.e. number of inputs).

For any non-negative integer $n$, let $\tau_n$ denote the function symbols in $\tau$ of arity $n$. A constant is a function symbol of arity $0$.

Let $\tau$ be a type, and let $X$ be a non-empty set of symbols, representing the variable symbols. (For simplicity, assume $X$ and $\tau$ are disjoint.) Then the **set of terms** $T(X)$ of type $\tau$ over $X$ is the set of all well-formed strings that can be constructed using the variable symbols of $X$ and the constants and operations of $\tau$. Formally, $T(X)$ is the smallest set such that:
- $X \cup \tau_0 \subseteq T(X)$ — each variable symbol from $X$ is a term in $T(X)$, and so is each constant symbol from $\tau_0$.
- For all $n \ge 1$ and for all function symbols $f \in \tau_n$ and terms $t_1, \ldots, t_n \in T(X)$, we have the string $f(t_1, \ldots, t_n) \in T(X)$ — given $n$ terms $t_1, \ldots, t_n$, the application of an $n$-ary function symbol $f$ to them represents again a term.

The **term algebra** $\mathcal{T}(X)$ of type $\tau$ over $X$ is, in summary, the algebra of type $\tau$ that maps each expression to its string representation. Formally, $\mathcal{T}(X)$ is defined as follows:
- The domain of $\mathcal{T}(X)$ is $T(X)$.
- For each nullary function $f$ in $\tau_0$, $f^{\mathcal{T}(X)}()$ is defined as the string $f$.
- For all $n \ge 1$ and for each $n$-ary function $f$ in $\tau$ and elements $t_1, \ldots, t_n$ in the domain, $f^{\mathcal{T}(X)}(t_1, \ldots, t_n)$ is defined as the string $f(t_1, \ldots, t_n).$

A term algebra is called *absolutely free* because for any algebra $\mathcal{A}$ of type $\tau$, and for any function $g : X \to \mathcal{A},$ $g$ extends to a unique homomorphism $g^* : \mathcal{T}(X) \to \mathcal{A},$ which simply evaluates each term $t \in T(X)$ to its corresponding value $g^*(t) \in \mathcal{A}$. Formally, for each $t \in T(X)$:
- If $t \in X$, then $g^*(t) = g(t)$.
- If $t = f \in \tau_0$, then $g^*(t) = f^{\mathcal{A}}()$.
- If $t = f(t_1, \ldots, t_n)$ where $f \in \tau_n$ and $n \ge 1$, then $g^*(t) = f^{\mathcal{A}}(g^*(t_1), \ldots, g^*(t_n)).$
