## Intro
> 🔗 https://en.wikipedia.org/wiki/Stochastic_programming

In the field of [mathematical optimization](https://en.wikipedia.org/wiki/Mathematical_optimization "Mathematical optimization"), **stochastic programming** is a framework for [modeling](https://en.wikipedia.org/wiki/Mathematical_model "Mathematical model") [optimization](https://en.wikipedia.org/wiki/Optimization_\(mathematics\) "Optimization (mathematics)") problems that involve [uncertainty](https://en.wikipedia.org/wiki/Uncertainty "Uncertainty"). A **stochastic program** is an optimization problem in which some or all problem parameters are uncertain, but follow known [probability distributions](https://en.wikipedia.org/wiki/Probability_distribution "Probability distribution"). This framework contrasts with deterministic optimization, in which all problem parameters are assumed to be known exactly. The goal of stochastic programming is to find a decision which both optimizes some criteria chosen by the decision maker, and appropriately accounts for the uncertainty of the problem parameters. Because many real-world decisions involve uncertainty, stochastic programming has found applications in a broad range of areas ranging from [finance](https://en.wikipedia.org/wiki/Finance "Finance") to [transportation](https://en.wikipedia.org/wiki/Transportation "Transportation") to energy optimization.

Several stochastic programming methods have been developed:
- Scenario-based methods including sample average approximation
- Stochastic [integer programming](https://en.wikipedia.org/wiki/Integer_programming "Integer programming") for problems in which some variables must be integers
- [Chance constrained programming](https://en.wikipedia.org/wiki/Chance_constrained_programming "Chance constrained programming") for dealing with constraints that must be satisfied with a given probability
- [Stochastic dynamic programming](https://en.wikipedia.org/wiki/Stochastic_dynamic_programming "Stochastic dynamic programming")
- [Markov decision process](https://en.wikipedia.org/wiki/Markov_decision_process "Markov decision process")
- [Benders decomposition](https://en.wikipedia.org/wiki/Benders_decomposition "Benders decomposition")
