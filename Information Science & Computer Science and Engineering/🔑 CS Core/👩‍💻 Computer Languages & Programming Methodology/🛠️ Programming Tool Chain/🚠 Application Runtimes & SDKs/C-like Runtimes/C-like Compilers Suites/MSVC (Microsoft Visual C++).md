## Intro
> 🔗 https://en.wikipedia.org/wiki/Microsoft_Visual_C%2B%2B

**Microsoft Visual C++** (**MSVC**) is a compiler for the [C](https://en.wikipedia.org/wiki/C_\(programming_language\) "C (programming language)"), [C++](https://en.wikipedia.org/wiki/C%2B%2B "C++"), [C++/CLI](https://en.wikipedia.org/wiki/C%2B%2B/CLI "C++/CLI") and [C++/CX](https://en.wikipedia.org/wiki/C%2B%2B/CX "C++/CX") [programming languages](https://en.wikipedia.org/wiki/Programming_language "Programming language") by [Microsoft](https://en.wikipedia.org/wiki/Microsoft "Microsoft"). MSVC is [proprietary software](https://en.wikipedia.org/wiki/Proprietary_software "Proprietary software"); it was originally a standalone product but later became a part of [Visual Studio](https://en.wikipedia.org/wiki/Microsoft_Visual_Studio "Microsoft Visual Studio") and made available in both [trialware](https://en.wikipedia.org/wiki/Trialware "Trialware") and [freeware](https://en.wikipedia.org/wiki/Freeware "Freeware") forms. It features tools for [developing](https://en.wikipedia.org/wiki/Software_development "Software development") and [debugging](https://en.wikipedia.org/wiki/Debugging "Debugging") C++ code, especially code written for the [Windows API](https://en.wikipedia.org/wiki/Windows_API "Windows API"), [DirectX](https://en.wikipedia.org/wiki/DirectX "DirectX") and [.NET](https://en.wikipedia.org/wiki/.NET_Framework ".NET Framework").

Many [applications](https://en.wikipedia.org/wiki/Application_software "Application software") require [redistributable](https://en.wikipedia.org/wiki/Redistributable "Redistributable") Visual C++ runtime library packages to function correctly. These packages are frequently installed separately from the applications they support, enabling multiple applications to use the package with only a single installation. These Visual C++ redistributable and runtime packages are mostly installed for standard [libraries](https://en.wikipedia.org/wiki/Library_\(computing\) "Library (computing)") that many applications use.
